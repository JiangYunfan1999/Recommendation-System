# Recommendation-System
Recommendation algorithms on MovieLens 1M dataset

## What it is
My undergraduate thesis! 

本科毕业论文（又名，本科毕业の学术垃圾）

**Title**: 基于数据稀疏性问题的协同过滤算法研究与应用

**Content**：在MovieLens数据集上实现三种推荐算法：
1. 基于用户的协同过滤（UBCF）
2. 基于矩阵分解的协同过滤（MFBCF）
3. 基于深度学习的协同过滤（DLBCF）
